# Department Management System

A **Django-based Department Management System** that allows admins to manage company departments.  
It includes basic **CRUD operations** and a **user authentication system** (login & signup).   

---

## Features
-  User Authentication (Signup, Login, Logout)
-  Manage Departments (Add, Update, Delete, View)
-  Organized Dashboard for department listings
-  Responsive UI with Bootstrap 5

---

## 🛠️ Tech Stack
- **Backend**: Django, Python
- **Frontend**: HTML, CSS, Bootstrap
- **Database**: MySQL
